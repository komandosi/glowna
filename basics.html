Kontener - div

Kontenera nie widać, ale pozwala na organizację treści. Automatycznie ma on maksymalną dostępną szerokość.

Tekst w kontenerze
<div>Tekst w kontenerze</div>

Akapit - p

Akapitu nie widać, jest podobny do kontenera - div, ale służy do umieszczania w nim akapitu tekstu.

Tekst w akapicie.

<p>Tekst w kontenerze</p>

Link - odnośnik

Link pozwala na przejście do innej strony.

Główna strona <a href="index.html">Główna strona</a>

Style - wygląd

Arkusz stylu pozwala odzielić opis wyglądu strony od jej treści.

<style type="text/css"> ... treść styli ... </style>

Klasy

Klasę (class) możemy dodawać do każdego tagu html. Pozwala ona odnieść się do tego elementu np podczas określania wyglądu.

Tekst w kontenerze - czerwony
Tekst w kontenerze - zielony
<style type="text/css"> .red-text { color: red; } .green-text { color: green; } </style> <div class="red-text">Tekst w kontenerze - czerwony</div> <div class="green-text">Tekst w kontenerze - zielony</div>
